# fourcolor-lean
Lean 4 port of Georges Gonthier et al.'s formal Four Color Theorem proof, translated from Rocq/Coq with most of the initial porting work generated by Aristotle. Upstream Rocq source: <https://github.com/rocq-community/fourcolor>.
Setup: install `elan`, ensure the repo uses the pinned toolchain in `lean-toolchain`, then run:

```powershell
lake exe cache get
lake build
```

Math overview: see [`FourColor/README.md`](FourColor/README.md).
